# Movie Rental System
Java console application for managing movie rentals using MySQL database.

## Tech Stack
Java, MySQL, JDBC, OOP

## Features
- Add/Delete/Update movie records
- Customer registration 
- Issue & Return movie with due date calculation
- Search movies by name/genre
- Exception handling for invalid inputs

## How to Run
1. Create MySQL database `moviedb`
2. Run the SQL script in /database folder
3. Compile: javac Main.java
4. Run: java Main
